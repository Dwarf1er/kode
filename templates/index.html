Welcome to Kode!

Kode is a Turing-complete programming language based on natural language. The goal of this project is to enable kids to solve mathematical equations without having to use complex symbols. However, for advanced users Kode can do much more!


Playground

This is where you can write code using the Kode language! It is a live environment where you can test and run your code. Don't forget that Kode is a complete programming language, explore and test the limits!

Examples

This is where you will find examples of code written with Kode. You can load them and run them in the playground to learn from them. Look at the Rule 110 example to see the proof that Kode is Turing-complete!
